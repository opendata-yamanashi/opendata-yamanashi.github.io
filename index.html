山梨県オープンデータAPIプロジェクト

このサイトについて

山梨県オープンデータカタログサイトで公開されているオープンデータを Web API で提供しているサイトです。
開発は山梨県内の有志エンジニアで行っています。

利用にあたっては「山梨県オープンデータサイト利用規約」をご確認ください。

山梨県オープンデータサイト利用規約

温泉利用施設API

山梨県内の温泉利用施設情報を提供

出典：「山梨県公共温泉利用施設一覧（H30.12.1現在）」（山梨県）

API documentation GitHub

犯罪オープンデータAPI

山梨県警察で認知した犯罪情報を提供

出典：山梨県警察ウェブサイト

API documentation GitHub

日本の主な山岳一覧 (国土地理院データ) API

日本の主な山岳の情報(山名, 都道府県, 所在等, 標高, 緯度経度, 三角点名等)が取得できるAPI

出典：「日本の主な山岳一覧」（国土地理院）

API documentation GitHub
